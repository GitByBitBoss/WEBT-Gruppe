1. Background

DriveWell offers car, motorcycle, and truck training. Theory lessons rely heavily on paper. Students expect mobile learning tools; instructors need reporting. Existing tools are siloed and insufficient.

2. Project Rationale

A learning platform will provide a digital question bank, progress tracking, simulations, and teacher dashboards.

3. Special Features / Challenges

  • Large, versioned question bank
  • Role-based access (students, teachers, admin)
  • Mobile-first experience

4. Implementation Note (Skills)

CMS for course content; responsive frontend; Symfony backend for questions/results; databases (users, progress); API integration; Security; SEO; CI/CD.
